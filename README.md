# sistema-chamados-helpdesk
# 💻 Sistema de Chamados (Help Desk)

Projeto simples de um sistema de abertura e gerenciamento de chamados, desenvolvido com HTML, CSS e JavaScript.  

O objetivo é simular um ambiente de suporte técnico, permitindo registrar, acompanhar e atualizar solicitações de usuários.

---

## 🚀 Funcionalidades

- ✅ Abertura de chamados  
- ✅ Definição de prioridade (Baixa, Média, Alta)  
- ✅ Listagem de chamados  
- ✅ Atualização de status:
  - Aberto
  - Em andamento
  - Resolvido  
- ✅ Armazenamento de dados no navegador (LocalStorage)  

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  
- LocalStorage
